Arrays in C

Arrays store multiple values of the same type in contiguous memory locations. Learn fixed-size arrays, memory layout, traversal techniques, pointer relationships, 2D matrices, multi-dimensional arrays, and more.

1. What is an Array?

A fixed-size collection of similar elements stored sequentially in memory.

  • Array size is fixed when declared.
  • Elements must be of the same data type.
  • Index starts from 0.
  • No automatic bounds checking.
  • Fast random access: O(1).
  • Excellent cache performance due to sequential storage.

üìå Arrays Store Elements in Contiguous Memory

The most important property of a C array is that all elements are stored in contiguous (continuous) memory locations. Each element sits immediately next to the previous one in RAM.

  • Address difference between two elements = size of data type.
  • Allows pointer arithmetic: address = base + index * sizeof(type)
  • CPU caching becomes more efficient.
// Example: int takes 4 bytes
int a[5] = {10, 20, 30, 40, 50};

// Memory layout (contiguous)
1000 ‚Üí 10
1004 ‚Üí 20
1008 ‚Üí 30
1012 ‚Üí 40
1016 ‚Üí 50

2. Declaring & Initializing Arrays

Different ways to create and initialize arrays.

// Declaration only ‚Üí garbage values
int a[5];

// Declaration + Initialization
int b[5] = {10, 20, 30, 40, 50};

// Partial initialization ‚Üí remaining become 0
int c[5] = {1, 2}; 
// c = {1, 2, 0, 0, 0}

// Automatic size detection
int d[] = {4, 5, 6};   // size = 3

// All zeros
int z[10] = {0};

3. Accessing Array Elements

Using index numbers to read/write values.

int numbers[3] = {10, 20, 30};

printf("%d", numbers[0]);  // 10
numbers[2] = 99;
printf("%d", numbers[2]);  // 99

Input Example

for(int i = 0; i < 5; i++) {
    scanf("%d", &numbers[i]);
}

4. Traversing Arrays

Looping through array elements.

int arr[5] = {1, 2, 3, 4, 5};

for(int i = 0; i < 5; i++) {
    printf("%d ", arr[i]);
}

Reverse Traversal

for(int i = 4; i >= 0; i--) {
    printf("%d ", arr[i]);
}

5. Common Array Operations

Standard operations performed using loops.

  • Sum
  • Maximum & minimum
  • Linear search
  • Copying arrays
  • Frequency counting

Example: Sum

int sum = 0;
for(int i = 0; i < 5; i++)
    sum += arr[i];

Example: Maximum

int max = arr[0];
for(int i = 1; i < 5; i++)
    if(arr[i] > max)
        max = arr[i];

6. 2D Arrays (Matrix)

Stored in row-major contiguous memory.

int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

printf("%d", matrix[1][2]); // 6

Row-Major Memory Layout

1 2 3 4 5 6

Traversing 2D Array

for(int i=0; i<2; i++) {
    for(int j=0; j<3; j++) {
        printf("%d ", matrix[i][j]);
    }
    printf("\n");
}

7. Multi-Dimensional Arrays

Arrays of arrays stored contiguously dimension by dimension.

int arr3D[2][3][4];  
// 24 integers stored contiguously in memory

8. Arrays & Pointers Relationship

Arrays behave like constant pointers.

  • a ‚Üí base address
  • a == &a[0] (value same, type different)
  • Pointer arithmetic is allowed
int a[5] = {10,20,30,40,50};

printf("%p", a);       // base address
printf("%p", &a[0]);   // same

printf("%d", *(a + 2));  // 30

Cannot change base pointer:

// ERROR
a = a + 1;

9. Dynamic Arrays (malloc)

Used to overcome fixed-size limitation.

int *ptr = malloc(5 * sizeof(int));

for(int i=0;i<5;i++)
    scanf("%d", &ptr[i]);

free(ptr);
