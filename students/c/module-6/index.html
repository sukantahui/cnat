Module 6 — Arrays

Arrays in C

Arrays store multiple values of the same type. Learn 1D, 2D and multi-dimensional arrays, memory layout, and common operations.

1. What is an Array?

A collection of similar data elements stored in contiguous memory.

  • Array elements share the same data type.
  • Elements are accessed using index numbers.
  • Index starts from 0.
  • Stored in continuous (contiguous) memory blocks.
int marks[5];

2. Declaring & Initializing Arrays

Initialize arrays during or after declaration.

// Declaration
int a[5];

// Declaration + Initialization
int b[5] = {10, 20, 30, 40, 50};

// Partial initialization
int c[5] = {1, 2};  // others become 0

// Automatic size
int d[] = {4, 5, 6};

3. Accessing Array Elements

Use the index to read/write array values.

int numbers[3] = {10, 20, 30};

printf("%d", numbers[0]);  // 10
numbers[2] = 99;

printf("%d", numbers[2]);  // 99

4. Traversing an Array

Loop through all elements.

int arr[5] = {1, 2, 3, 4, 5};

for(int i = 0; i < 5; i++) {
    printf("%d ", arr[i]);
}

5. Common Array Operations

Simple tasks performed using loops.

  • Sum of array elements
  • Max/Min value
  • Linear search
// Sum of array
int sum = 0;
for(int i = 0; i < 5; i++) {
    sum += arr[i];
}
printf("Total = %d", sum);

6. 2D Arrays (Matrix)

Useful for tables and matrices.

int matrix[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};

printf("%d", matrix[1][2]); // 6
    
// Traversing 2D
for(int i=0; i<2; i++) {
    for(int j=0; j<3; j++) {
        printf("%d ", matrix[i][j]);
    }
    printf("\n");
}

7. Multi-Dimensional Arrays

3D, 4D arrays used in advanced computing.

int arr3D[2][3][4];
